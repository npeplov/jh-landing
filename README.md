# jh-landing
Landing for coming soon page. Stack: vite + react + MUI

Демо
https://npeplov.github.io/jh-landing/

#### Установка с помощью Node
```
git clone git@github.com:JuniorHubProject/jh-landing.git
cd jh-landing
npm i
```

Запуск
```
npm run dev
```

Сборка для деплоя
```
npm run build
```
